# CSE-CIC-IDS2018-modified
Datasets de tráfego de rede baseados no CSE-CIC-IDS2018 e modificados para pesquisa em Redes Definidas por Software (SDN). Inclui tráfego sintético e duplicações para testes de detecção de ataques DDoS.

Network traffic datasets based on CSE-CIC-IDS2018 and modified for research in Software-Defined Networking (SDN). Includes synthetic traffic and duplicates for DDoS attack detection testing.
